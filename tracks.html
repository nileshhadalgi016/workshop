Beginner Path Level 1

Who it's for: New to electronics / programming. No prior hardware experience required.

Goal: Comfort with microcontroller basics, reading sensors, and building simple interactive prototypes.

Outcomes

  • Understand digital vs analog I/O
  • Wire & read common sensors (temperature, light, motion)
  • Build a simple autonomous gadget (LED feedback + sensor trigger)
  • Confident using serial monitor for debugging

Modules

  1. Arduino Basics 3h • Live Build
  2. Sensors & Inputs 3h • Labs
  3. Simple Robotics (Coming Soon) 4h • Project

Intermediate Path Level 2

Who it's for: Makers who finished Beginner Path or have basic Arduino knowledge, wanting Linux + connectivity.

Goal: Move from microcontrollers to embedded Linux, interface multiple devices, and capture structured data.

Outcomes

  • Comfortable with Raspberry Pi OS & shell basics
  • Use Python to control GPIO & handle events
  • Integrate I2C / SPI modules
  • Log & visualize sensor data locally

Modules

  1. Raspberry Pi Fundamentals 4h • Guided
